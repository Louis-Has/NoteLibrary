微服务
- 一种系统架构的设计方式，**将复杂的业务拆分成多个微小的服务**
- 为了服务可以单独去运行和部署
- 服务之间采用 RPC 通信

分布式
- 一种系统的部署方式，将服务部署到多台机器上
- **分摊机器的负载压力，提高吞吐量**
- 可以采用集群、主备等方式部署
